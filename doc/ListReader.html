ListReader

Class: ListReader

Defined in: src\ListReader.coffee

Overview

ListReader reads in a text file and gives an array of its lines

Examples:

new ListReader().readList "data/list.txt", (err, arr) ->
  console.log el for el in arr

Instance Method Summary

  • ~ (void) readList(path, callback) Reads in a list from a text file and returns the array created.

Constructor Details

# (void) constructor(path)

Creates a new ListReader. You can preset the file path as an argument to the constructor or give it when calling readList The path is relative to the working directory where Node is run.

Parameters:

  • (path) ( String ) — The path to the file that is to be converted

Instance Method Details

~ (void) readList(path, callback)
~ (Promise) readList(path)

Reads in a list from a text file and returns the array created. When the file is read or an error occurs, the callback is called. If no callback is given, a promise for the array is returned. The path is relative to the working directory where Node is run.

Please note that the reader discards the first character it encounters as it assumes it to be a Byte Order Mark.

Throws:

  • ( ? ) — No path set

Overloads:

~ (void) readList(path, callback)

Parameters:

  • (path) ( String ) — The file path of the file. If not set, the path from the constructor will be used
  • callback ( function ) — The callback (err, result) with err being any errors from fs.createReadStream and result the Array of lines from the file

~ (Promise) readList(path)

Parameters:

  • (path) ( String ) — The file path of the file. If not set, the path from the constructor will be used
